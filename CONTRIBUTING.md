# Contributing

## Как да допринесеш
1. Направи промените директно в `index.html`.
2. Тествай ръчно по чеклиста в [`qa-docs/TEST_CASES.md`](qa-docs/TEST_CASES.md) — особено филтри, количка и checkout.
3. Провери, че комбинирани филтри (search + type + brand) продължават да работят коректно заедно.
4. Commit съобщенията да описват кратко какво и защо се променя.

## Docs
При добавяне на нова функционалност, обнови съответно:
- [`README.md`](README.md)
- [`qa-docs/TEST_CASES.md`](qa-docs/TEST_CASES.md)
- [`CHANGELOG.md`](CHANGELOG.md)

# Тест кейсове — КаниКар

Легенда: **P** = Priority (P1 критичен, P2 висок, P3 среден)

## Търсене и филтри

| ID | Стъпки | Очакван резултат | P |
|---|---|---|---|
| TC-01 | Въведи размер гума (напр. "6.00-9") в търсачката | Резултатите се филтрират до съвпадащите размери | P1 |
| TC-02 | Избери стойност от филтър „Тип машина" | Продуктовият списък показва само съответните продукти | P1 |
| TC-03 | Избери „Конструкция" филтър | Списъкът се филтрира коректно по конструкция | P1 |
| TC-04 | Избери „Наличност" (напр. само налични) | Продуктите извън наличност се скриват/маркират | P2 |
| TC-05 | Кликни chip за марка в „Марка" филтъра | Списъкът се филтрира по избраната марка, chip-ът се маркира active | P2 |
| TC-06 | Комбинирай 2+ филтъра едновременно | Резултатите отговарят на пресечната точка на всички избрани критерии | P1 |
| TC-07 | Натисни „✕ Изчисти" (resetFilters) | Всички филтри и търсене се нулират, показва се пълният каталог | P1 |

## Детайлна карта на продукт (Modal)

| ID | Стъпки | Очакван резултат | P |
|---|---|---|---|

## Количка и Checkout

| ID | Стъпки | Очакван резултат | P |
|---|---|---|---|
| TC-18 | Отвори количката (openCart) с 0 продукта | Checkout бутонът е disabled | P1 |
| TC-19 | Добави продукт и отвори количката | Продуктът и коректната обща сума се показват | P1 |
| TC-20 | Натисни ✕ (closeCart) | Количката се затваря, съдържанието се запазва | P2 |
| TC-21 | Натисни „Checkout" (openCheckout) с продукти в количката | Стартира checkout флоу с коректни данни | P1 |

## Auth / Owner достъп

| ID | Стъпки | Очакван резултат | P |
|---|---|---|---|
| TC-22 | Кликни auth бутон (openAuthModal) като гост | Отваря се модал за вход/регистрация | P1 |
| TC-23 | Опитай достъп до owner функции без логин | Достъпът е отказан / функциите не се виждат | P1 |
| TC-24 | Влез с реалния admin акаунт (email+парола през login формата) | Admin-специфични контроли (⚙ АДМИН ПАНЕЛ) стават достъпни | P1 |
| TC-28 | ~~ownerLogin()~~ бе премахнат 2026-07-24 (беше admin bypass без парола) — провери в конзолата, че функцията вече не съществува и forge-нат `localStorage`/cookie не дава admin достъп | И двата опита за bypass се провалят; автоматизиран регресионен тест: `tests/security.spec.js` | P1 |
